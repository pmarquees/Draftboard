# Draftboard Design System

Welcome to the Draftboard component library. This Storybook documents all reusable UI components used throughout the application.

## Overview

Draftboard uses a modern stack built on:

- **React 19** with Next.js 15
- **Tailwind CSS v4** for styling
- **Radix UI** for accessible primitives
- **Class Variance Authority (CVA)** for component variants
- **Lucide React** for icons

## Usage Guidelines

### Import Pattern

All UI components should be imported from `~/components/ui/`:

```tsx
import { Button } from "~/components/ui/button";
import { Card, CardHeader, CardTitle, CardContent } from "~/components/ui/card";
```

### Class Merging

Use the `cn()` utility for conditional class merging:

```tsx
import { cn } from "~/lib/utils";

<Button className={cn("custom-class", isActive && "active")} />
```

### Accessibility

All components are built on Radix UI primitives, ensuring:
- Full keyboard navigation
- Screen reader support
- Focus management
- ARIA attributes
